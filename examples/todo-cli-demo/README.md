# todo-cli demo（已完成的示範跑，僅供參考）

這是 DevLoop 第一次端對端行為驗證的完整存底 —— **範本 / 參考資料，不是
可執行的活動狀態**，不影響 `devloop/` 或 `workspace/` 目前的實際運作。

- `workspace-todo-cli/` — 當時做出來的 todo.js + todo.test.js（3 個 human 任務做完、
  8 個 ai 發想任務待核准時的狀態）
- `devloop-snapshot/` — 對應當時的 `devloop/GOAL.md` / `TASKS.md` / `PROGRESS.md` /
  `LESSONS.md` 快照，可用來參考 GOAL.md 該怎麼寫、任務描述該多細、
  PROGRESS.md 該怎麼記錄

想重看示範怎麼跑起來的，可另開目錄把 `workspace-todo-cli/` 複製回
`workspace/todo-cli/`、把 `devloop-snapshot/*.md` 複製回 `devloop/*.md`。
